# RixAI

Chrome-расширение для Google NotebookLM. Добавляйте страницы, видео и ссылки в ноутбуки одним кликом.

---

## Русский

**RixAI** — расширение для браузера (Chrome, Edge, Yandex). Оно подключается к Google NotebookLM и позволяет быстро добавлять в ноутбуки текущую страницу, YouTube-плейлисты и каналы, массовый импорт ссылок и открытых вкладок.

### Возможности

- Добавление текущей страницы или видео в выбранный ноутбук одним кликом
- Импорт всех видео из плейлиста или канала YouTube
- Массовый импорт: вставка списка ссылок или импорт открытых вкладок
- Управление источниками: массовое удаление в NotebookLM
- Несколько аккаунтов Google, русский и английский интерфейс

### Установка

1. Скачайте или клонируйте репозиторий
2. Откройте `chrome://extensions/`
3. Включите «Режим разработчика»
4. Нажмите «Загрузить распакованное расширение» и выберите папку с расширением

### Использование

1. Войдите в [NotebookLM](https://notebooklm.google.com)
2. Откройте расширение, выберите ноутбук и нажмите «Добавить в ноутбук»
3. Для массового импорта откройте страницу расширения через кнопку «Массовый импорт»

## English

**RixAI** is a browser extension (Chrome, Edge, Yandex) that works with Google NotebookLM. Add the current page, YouTube playlists and channels, or bulk-import links and open tabs into your notebooks.

### Features

- Add current page or video to a selected notebook with one click
- Import all videos from a YouTube playlist or channel
- Bulk import: paste a list of URLs or import open tabs
- Manage sources: bulk delete in NotebookLM
- Multiple Google accounts; Russian and English interface

### Installation

1. Download or clone this repository
2. Open `chrome://extensions/`
3. Enable Developer mode
4. Click «Load unpacked» and select the extension folder

### Usage

1. Sign in at [NotebookLM](https://notebooklm.google.com)
2. Open the extension, choose a notebook, and click «Add to notebook»
3. For bulk import, open the extension page via the «Bulk Import» button
